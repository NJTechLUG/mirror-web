<h1>Archlinux</h1>
<h2>地址</h2>
https://mirrors.ustc.edu.cn/archlinux/

<h2>说明</h2>
Arch Linux 软件源

<h2>收录架构</h2>
i686, x86_64

<h2>使用说明</h2>
编辑 /etc/pacman.d/mirrorlist ，在文件的最顶端添加</br>
<code>
Server = https://mirrors.ustc.edu.cn/archlinux/$repo/os/$arch
</code></br>
国内用户，推荐另外使用 Arch Linux CN 的源，请参考 Arch Linux CN 源使用帮助
